Pong Clone (C++ / SFML)

.A simple Pong clone built in C++ using SFML.
.This project focuses on collision detection, input handling, and basic 2D rendering.

Features

.Two-player local gameplay

.Paddle movement with keyboard input

.Ball physics and collision detection

.Score tracking

Controls

.Player 1    
- W = UP  
- S = DOWN  

.Player 2
- O = UP
- L = Down
